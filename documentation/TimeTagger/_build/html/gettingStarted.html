Getting Started¶

Windows users¶

  1. Make sure the TimeTagger software is installed and the TimeTagger connected to your computer (see the Windows quickstart card).

  2. Open a command shell and cd to the examples folder on the USB stick

  3. Start an ipython shell

  4. Run the test.py script by entering run test, this will create an instance time_tagger that represents the device.

  5. Check the channel jitter calibration by ...

  6. Start the builtin test signal by … this will enable an about 3 MHz square wave applied internally to all input channels

  7. Create a simple countrate on the first two channel … by entering …

  8. Upon running of count_rate.getData() will return the average countrate since its instantiation

  9. Create a crosscorrelation …

  10. Connect an external signal (e.g. a 1 MHz square wave, 1V).

    Danger

    Make sure that the frequency of the external signal does not exceed the maximum time tag rate of ~10 MHz. Deactivate the builtin test signal.

  11. Adjust the triggerlevel see how the rate changes.
